# Unblock Music

一个解锁网易云音乐的小工具

## To Download

Down [releases](https://github.com/zenoslin/unblock-music/releases)

## To Build

```js
git clone https://github.com/zenoslin/unblock-music.git
cd unblock-music
npm install
npm run start
```
